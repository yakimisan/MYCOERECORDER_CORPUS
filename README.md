# [コーパス名]

## 📝 概要

本データは、音声合成エンジン **MYCOEIROECORDER** に利用するために作成・調整された学習用コーパスデータです。

## 🎯 利用目的

MYCOEIROECORDER を用いた音声モデルの学習を目的としています。

## 📜 ライセンス

本コーパスは、**パブリックドメイン (Public Domain)** として配布されます。

利用にあたり、特別な制限や制約はありません。

## 🔧 作成ベース

このコーパスは、以下のプロジェクトをベースとして作成・管理されています。

* **Coeiroink Corpus Manager**
    * [https://github.com/shirowanisan/coeiroink-corpus-manager](https://github.com/shirowanisan/coeiroink-corpus-manager)

## 📚 使用データ

本コーパスの作成には、以下の既存のコーパスデータが使用されています。

* **MANAコーパス**
* **ITAコーパス**
    * [https://github.com/mmorise/ita-corpus](https://github.com/mmorise/ita-corpus)
